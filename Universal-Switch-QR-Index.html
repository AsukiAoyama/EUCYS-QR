The Universal-Switch

WELCOME
Here you can read all the important information about my project: The Universal-Switch.
SUMMARY
The project is about the development of a new smart-home device, which could replace entire smart-home installations in a cost-effective way and for less renovation or wiring work. The Universal-Switch combines various switches, buttons and dimmers into a single light-switch sized device. The Universal-Switch can be built in inside a flush-mounted box and uses the preinstalled wiring to control the respective consumer-devices. The user can control the switch using an app or by touching the touch-surface of the device. The possibility of changing the switching behavior and combining actuator and sensor in one device makes the Universal-Switch suitable for every existing and new smart-home.
HARDWARE
The Universal-Switch is controlled by the ESP32-microcontroller. The built-in Wi-Fi module of the microcontroller is used to host the web-app within the home's Wi-Fi network. DIP-switches are used to manually set the desired function of the respective device. To switch the 230V AC present in the house installation, I currently use mechanical relay outputs. Simple transistor driver-circuits were designed to switch the relays safely and to control the backlight-LEDs of the touch-surface. These LEDs show the user the currently active areas on the touch-surface. The touch function of the Universal-Switch is implemented using capacitive TTP223 touch-sensors
Software
The project's software is divided into two main groups: control-software and web-software. The control-software's tasks involve recognizing user inputs on the touch-surface and accordingly switching the relay outputs. Additionally, the control-software is used to launch and manage the web-app within the home's own Wi-Fi network. This web-app resides within the home's own Wi-Fi and is managed by the ESP32-microcontroller and its control-software. The web-app can be accessed via a smartphone, allowing mobile control and monitoring of the Universal-Switch. The programming languages HTML, CSS, and JavaScript were used to develop the web-software, with Visual Studio Code serving as the development environment.
